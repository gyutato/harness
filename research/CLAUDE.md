# research 하네스

## 하네스: 논문 심층 리서치

**목표:** 논문(단일/다중/주제 서베이)을 심층 독해하여 상세 설명 + 비판적 인사이트가 담긴 보고서를 생성한다.

**트리거:** 논문 URL·PDF·DOI·제목 제공, "논문 분석", "이 논문 읽어줘", "서베이", "인사이트 뽑아줘" 등 논문 기반 리서치 요청 시 `research-orchestrator` 스킬을 사용하라. 단순 질의응답이나 코드 작성은 직접 응답.

**실행 모드:** 에이전트 팀 (교차 검증 + 도구 권한 실시간 피드백). paper-hunter만 서베이 모드에서 서브 에이전트로 단독 호출하는 하이브리드.

**에이전트 구성:** research-lead(리더/오케스트레이터) / paper-hunter(Phase 1.5, 서브) / deep-reader(팀원) / context-scout(팀원) / insight-analyst(팀원)

**작업 디렉토리 규약:** 실행 시 현재 디렉토리에 `_workspace/` 생성. 중간 산출물은 보존하여 후속 작업에 재활용.
